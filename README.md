# sistemafrota

> A Vue.js project

## Build Setup

``` bash
# Instala todas as dependências do package.json
npm install

# Roda todos os testes [ INTEGRATION / CONTRACT / UNIT ]
npm run test

# Inicia a aplicação
npm start
```
